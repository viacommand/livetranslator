# 🌍 Use Me Translator

A modern, responsive web-based language translator built using **HTML, CSS, and JavaScript**. The application provides a clean glassmorphism interface with an animated 3D background and allows users to translate text into multiple languages.

---

## 📌 Features

- 🌐 Translate text into multiple languages
- 🎨 Modern Glassmorphism UI
- ✨ Animated 3D Background using Three.js
- 📱 Fully Responsive Design
- ⚡ Fast and Lightweight
- 📝 Simple and User-Friendly Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Three.js
- Google Translate API / Translation API (if configured)

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/use-me-translator.git
```

### 2. Open the Project

Navigate to the project folder.

```bash
cd use-me-translator
```

### 3. Run the Project

Simply open **index.html** in your browser.

Use **VS Code Live Server** for a better development experience.
